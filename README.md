# CS779
This repo includes assignments of CS779 and the final Capstone Project.
The Capstone project focuses on building a Neural Machine Translation (NMT) system for translating En
glish to Hindi and Bengali using recurrent neural networks. A sequence-to-sequence architecture with
attention was implemented through multiple GRU and LSTM models. The final Bidirectional LSTM
with Attention achieved a chrF++ score of 0.106, ROUGE score of 0.235, and BLEU score of 0.016,
ranking 94th on the final leaderboard.
